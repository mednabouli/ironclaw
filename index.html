Why IronClaw

Everything the others left out

Built with Rust's zero-cost abstractions, every feature compiles in only when you need it.

🧩
WASM Plugin System
Ship tools as .wasm Component Model files. Sandboxed wasmtime execution with fuel metering, memory limits, and capability-gated host imports (HTTP, filesystem, env).
🤖
Agent Swarms
ReAct, CriticActor, Router, ChainOfThought, Planner, Swarm, and Workflow patterns built in. Delegate subtasks across a DAG of specialized agents with eval harness.
⚡
8 LLM Providers
Ollama, Claude, OpenAI, Groq, OpenRouter, Mistral, Together, and Cohere. Local-first by default. Automatic failover chain with health checks, retry, and circuit breaker.
📡
8 Channels
CLI, REST/SSE, Telegram, Discord, Slack, WebSocket, Matrix, and Webhook — each feature-flagged, zero binary bloat when unused.
🔄
Hot-Reload Config
Edit ironclaw.toml while running. File watcher detects changes instantly via notify + arc-swap — no restart, no downtime.
📦
Embeddable Library
Use ironclaw-core as a library in your Tauri, Axum, or CLI app. 9 built-in tools, SQLite + Redis memory, and trait-based design means any component is swappable.

Quick Start

Up in 30 seconds

Install the binary, drop a config file, and run.

ironclaw.toml
# Minimal config — Ollama local + Telegram [agent] name = "IronClaw" system_prompt = "You are a helpful assistant." [providers] primary = "ollama" fallback = ["claude", "groq"] [providers.ollama] model = "llama3.3:70b" [providers.claude] api_key = "${ANTHROPIC_API_KEY}" [channels] enabled = ["telegram", "cli"] [channels.telegram] token = "${TELEGRAM_TOKEN}" [memory] backend = "sqlite"
Install
$ curl -fsSL https://ironclaw.dev/install.sh | bash
Start all channels
$ ironclaw start ✅ Provider: ollama — healthy ✅ Channel: telegram — connected 🦀 Startup: 57ms
Interactive TUI
$ ironclaw chat $ ironclaw run "summarize README.md" $ ironclaw batch prompts.jsonl --concurrency 8 $ ironclaw plugin install https://ironclaw.dev/plugins/crypto-price.wasm
Use as a library
# Cargo.toml [dependencies] ironclaw-core = "0.1" ironclaw-agents = "0.1"
